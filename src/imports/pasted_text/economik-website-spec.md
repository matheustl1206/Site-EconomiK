# ECONOMIK WEBSITE

# CREATIVE DIRECTION + UI/UX + GSAP ANIMATION SPECIFICATION

---

# EXPERIENCE GOAL

The user must not feel they are browsing a university website.

The user must feel they are entering a premium economic intelligence platform.

Keywords:

* Intelligence
* Precision
* Analysis
* Confidence
* Growth
* Strategy

The entire website should communicate:

"We observe before we decide."

* Generic gradients

# SCROLL PHILOSOPHY

Scrolling should feel like uncovering a report.

Each section reveals information progressively.

User should feel:

"I am discovering insights."

Not:

"I am browsing pages."
