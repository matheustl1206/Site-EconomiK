# ECONOMIK WEBSITE - MASTER DEVELOPMENT BRIEF

## PROJECT OVERVIEW

Develop a premium institutional website for ECONOMIK (Junior Economics Company).

The website must not look like a traditional university junior company.

The perception goal is:

"A modern economic intelligence consultancy that transforms data into strategic decisions."

The website should combine:

* Premium design
* High-end animations
* Storytelling
* Lead generation
* SEO optimization
* Fast performance
* Responsive experience

The website should feel similar to modern consulting firms, fintechs and data intelligence companies.
